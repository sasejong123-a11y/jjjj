# 로또 번호 추천 사이트 프로젝트

## 1. 프로젝트 개요

이 프로젝트는 사용자에게 로또 번호를 추천해주는 웹 애플리케이션입니다. 사용자가 버튼을 클릭하면, 시스템은 1부터 45까지의 숫자 중에서 6개의 유니크한 번호를 무작위로 선택하여 화면에 보여줍니다. 최신 웹 기술과 디자인 트렌드를 반영하여 시각적으로 매력적이고 사용자 친화적인 인터페이스를 제공하는 것을 목표로 합니다.

## 2. 디자인 및 기능 구현 계획

### 스타일 및 디자인 (style.css)

- **레이아웃**:
    - 전체 페이지를 아우르는 부드러운 그라데이션 배경을 적용하여 깊이감과 입체감을 더합니다.
    - 콘텐츠를 중앙에 배치하고, `box-shadow`를 활용하여 입체적인 느낌의 카드를 디자인합니다.
    - `noise` 텍스처를 배경에 추가하여 고급스러운 느낌을 줍니다.
- **색상**:
    - 활기차고 긍정적인 느낌을 주는 다채로운 색상 팔레트를 사용합니다.
    - 생성된 각 로또 번호에 그라데이션 색상을 적용하여 시각적 구분을 명확히 합니다.
- **타이포그래피**:
    - 가독성이 좋은 모던한 글꼴을 선택합니다.
    - 제목, 본문, 버튼 등 요소에 따라 글꼴 크기와 굵기를 조절하여 정보의 위계를 명확하게 전달합니다.
- **애니메이션**:
    - 버튼 클릭 시 부드러운 전환 효과를 적용합니다.
    - 로또 번호가 생성될 때 나타나는 애니메이션을 추가하여 동적인 느낌을 줍니다.

### 기능 구현 (main.js)

- **번호 생성**:
    - '번호 생성' 버튼 클릭 시, 1부터 45까지의 숫자 중에서 중복되지 않는 6개의 숫자를 무작위로 추출합니다.
    - 추출된 번호는 오름차순으로 정렬하여 사용자에게 보여줍니다.
- **동적 UI 업데이트**:
    - JavaScript를 사용하여 생성된 번호를 HTML에 동적으로 추가하고, 각 번호에 스타일을 적용합니다.
    - 이전에 생성된 번호가 있다면, 새로운 번호를 생성하기 전에 화면에서 제거합니다.

### HTML 구조 (index.html)

- 시맨틱 HTML 태그를 사용하여 웹 접근성을 준수하고 코드의 의미를 명확히 합니다.
- 메인 컨테이너 안에 제목, 번호가 표시될 영역, 그리고 '번호 생성' 버튼을配置합니다.
- CSS 파일과 JavaScript 파일을 적절히 연결합니다.

## 3. 현재 요청 작업 계획

1. `index.html` 파일의 콘텐츠를 로또 번호 추천 사이트의 기본 구조로 변경합니다.
2. `style.css` 파일을 업데이트하여 새로운 디자인 가이드라인에 맞는 스타일을 적용합니다.
3. `main.js` 파일을 업데이트하여 로또 번호 생성 및 UI 업데이트 로직을 구현합니다.
